# V10-1 Ticket 03：職業選擇背景故事

**狀態：** ✅ 已完成

**依賴：** 無（可獨立於 T01/T02 實作）

---

## 目標

職業選擇卡片目前只有數值加成與技能預覽，缺乏個性。加入各職業的「過去身份」背景故事，讓玩家在選職業時有情感投入。

---

## 各職業背景故事

| 職業 | backstory |
|------|-----------|
| 劍士 | 你是前線衛隊的老兵，一柄長劍百戰不敗。那次任務孤身深入廢墟追蹤魔物群，以一擋百，力竭倒下。身體仍記得一切——手握劍柄的瞬間，熟悉的重量回來了。 |
| 弓手 | 你是遊走四方的賞金獵人，沒有歸屬，靠著長弓和快腿討生活。廢墟中那場遭遇讓你栽了跟頭，對手不按常理出牌。這次，你會記取教訓。 |
| 法師 | 你是學院出身的術士，因鑽研禁忌魔法遭驅逐。廢墟裡封印的古代知識讓你如獲至寶，卻也讓你付出代價。法力仍在，只需時間重新掌控。 |
| 聖騎士 | 你曾是聖堂的護衛騎士，誓言守護弱者。那場戰鬥中你以血肉之軀擋住同伴退路，卻讓自己陷入絕境。聖光仍在血脈中流淌——不是為了榮耀，而是一個承諾。 |

## 修改檔案

### `StaticData/ClassDef.swift`

新增欄位：

```swift
struct ClassDef {
    // ...existing fields...
    let backstory: String
}
```

### `Views/ClassSelectionView.swift`

職業卡片底部新增「你的過去」區塊，用 `Divider` 與加成資訊分隔：

```
┌────────────────────────────────┐
│ 職業名稱 + 技能預覽             │
│ 屬性加成                        │
│ ─────────────────────────────  │
│ 你的過去                        │
│ （斜體灰色，約 50 字背景故事）   │
└────────────────────────────────┘
```

```swift
// 卡片底部
Divider()
VStack(alignment: .leading, spacing: 4) {
    Text("你的過去")
        .font(.caption)
        .foregroundStyle(.secondary)
    Text(classDef.backstory)
        .font(.caption)
        .italic()
        .foregroundStyle(.secondary)
        .fixedSize(horizontal: false, vertical: true)
}
```
