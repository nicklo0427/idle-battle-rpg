### 金幣掉落範圍

- 各樓層金幣產出能否支撐鑄造配方的金幣需求
- 玩家不應卡在「無法攢夠金幣鑄造下一件裝備」

### 鑄造配方素材數量

- 採集速率（素材 / 小時）是否能合理支撐鑄造需求

## 平衡驗證方法

### 試算表驗證（上線前）

建立成長試算：
1. 初始裝備 → 荒野邊境 F1 首通 → 取得飾品配方
2. 採集 X 小時 + 出征 Y 次 → 攢夠材料 → 鑄造飾品
3. 裝備後戰力 → 挑戰 F2 勝率
4. 依此類推，確認每個「採集 → 鑄造 → 裝備 → 挑戰」循環的時間合理

### 實機測試（有 Debug Build）

使用現有開發模式工具：
- 「升至最高等級」快速測試高端裝備
- 「卸除所有裝備」測試低戰力體驗
- 短時間採集 / 出征測試獎勵速率

## 驗收標準

- [ ] 從初始狀態到首區 Boss 首通，總遊玩時間（AFK）在合理範圍（約 3–6 小時真實時間）
- [ ] 每個首通裡程碑前，玩家至少有 1–2 次完整「採集 → 鑄造 → 裝備」循環
- [ ] Boss 武器 farming 有意義（浮動範圍夠大，最優值值得追求）
- [ ] 素材 / 金幣供需平衡，不出現長期卡關
- [ ] 所有佔位值替換為正式數值，並在 `DungeonRegionDef` / `EquipmentDef` 加上對應注釋
