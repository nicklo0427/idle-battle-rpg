## 實作規範

### CraftRecipeDef 擴充

`StaticData/CraftRecipeDef.swift` 新增 12 筆 V2-1 配方，新增欄位：

```swift
struct CraftRecipeDef {
    // 現有欄位
    let key: String
    let outputEquipKey: String
    let materials: [(MaterialType, Int)]
    let goldCost: Int
    let durationSeconds: Int

    // 新增
    let unlockedByFloorKey: String?   // nil = 預設可見（V1 配方）
                                       // 非 nil = 需首通對應樓層才顯示
}
```

### CraftSheet 顯示邏輯

- V1 配方（`unlockedByFloorKey == nil`）：永遠顯示
- V2-1 配方：需 `DungeonProgressionService.isFloorCleared(...)` 為 `true` 才顯示
- 未解鎖配方：**不顯示**（而非顯示為灰化，避免 UI 過於複雜）

### 鑄造時長參考

| 裝備等級 | 時長 |
|---|---|

| 廢棄礦坑裝備 | 15–30 分鐘 |
| 古代遺跡裝備 | 30–60 分鐘 |
